# Computer-sends-alert-to-mobile-phone
Computer sends alert to mobile phone.After setting up the channel, press the blue button to alert the phone, text can be included.
Installation files and their instructions
“频率”-Channel
“附带文字”-Accompanying text
Please change the channel before use！
If you can't see the blue button, please extend the window downward

Why I made this
I often let AI agents (Codex, open claw.) work on tasks while I am away from my computer.
Sometimes the agent encounters something it cannot solve by itself, such as:
* CAPTCHA verification
* Login confirmation
* Human approval dialogs
* Security checks
The agent then waits for user input.
The problem is that I may not return to my computer for hours, causing the entire task to remain stalled.
Remote Bell was created to solve this problem.
When human intervention is required, the computer sends an alert to my phone and triggers a loud ringtone, even if the phone is locked and the app is running in the background.
This allows me to immediately return and help the AI continue its work.
